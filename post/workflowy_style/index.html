workflowy 自定义 css

By Seyee | created 2023-09-04 | 268 words | 1 minutes
workflowy

使用 WFx插件 可以全局修改 workflowy 的 css（包括移动端）

改了一个 todo 的样式，如果点击完成，则会在前面加上一个完成的图标，并且去掉官方的删除线样式。预览如下（上面是使用后，下面是使用前）：

普通的 bullets 的完成样式

/* 去掉删除线 */
 .done .content {
    opacity: 1 !important;
    text-decoration: none !important;
}
/* 前面加个图标，不想加可以去掉 */
/* 包含标签的已完成加上图标 */
.done .content:has(.contentTag)::before {
    content: '✅ '; 
}
/* 普通已完成加上图标 */
.done .content::before {
    content: '🕙 ';
}

.done.checkmark .content::before {
